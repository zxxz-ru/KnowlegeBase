Some command line commands to start working with Postgresql

Installation

 sudo apt-get install postgresql // install server
 sudo pg_lsclusters // use script ot install clusters
 dpkg -l | grep postgr // check if OK

Right after installation:

To open connection to psql must type: sudo -u postgres psql and enter sudo password.



Create Role for user:

CREATE ROLE name of user WITH PASSWORD 'password';

ALTER ROLE name of user WITH LOGIN;

ALTER ROLE name of user WITH CREATEDB

Have tried to do all in one statement for some reason it was impossible.



To import db from tar file:

psql postgres -U name of user -h 127.0.0.1; Must conect to server first.

CREATE DATABASE name of database; Create db.

\q Drop connection to server.

pg_restore -v -d name of database -U name of user -h host (I used localhost) path to tar file;
