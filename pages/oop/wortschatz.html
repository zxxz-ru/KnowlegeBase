OOP

Object-oriented programming is an approach to designing modular reusable software systems.

Class

the definition for the data format and available procedures for a given type or class of object;

A class is a blueprint or prototype from which objects are created

**КЛАСС** - это шаблон, предназначенный для создания объектов. Класс - это именованный тип сущности из предметной области, возможно, имеющая предка (суперкласс), определенная как набор полей и методов.

Object

In Dictionary: a material thing that can be seen and touched.

In OOP: a particular instance of a Class, where the object can be a combination of variables, functions, and data structures.

Объект — это именованная сущность, одновременно содержащая данные(свойства) и поведения(методы). Слово «одновременно» в данном случае определяет ключевую разницу между ООП и другими методологиями программирования. Например, при процедурном программировании код размещается в полностью отдельных функциях или процедурах.

Interface

An interface is a contract between a class and the outside world. When a class implements an interface, it promises to provide the behavior published by that interface.
