The Trick with tsc that it isn't globaly installed and thus is not in $PATH. To run tsc full relative path must be used. find -name tsc in root dir of Project give such result:
./node_modules/tsutils/node_modules/.bin/tsc
./node_modules/typescript/bin/tsc
./node_modules/.bin/tsc
./node_modules/tslint/node_modules/.bin/tsc
 I used ./node_modules/typescript/bin/tsc --outDir build workes for compilation and puts compiled files in ./build for some reason it doesn't work if I put outDir option in tsconfig file. My tsconfig was:

  },
