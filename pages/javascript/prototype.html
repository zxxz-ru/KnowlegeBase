Every object can have another object as its prototype. Then the former object inherits all of its prototype’s properties. An object specifies its prototype via the internal property [[Prototype]]. The chain of objects connected by the [[Prototype]] property is called the prototype chain. в Функции конструкторе в переменной prototype тот указатель на то что будет записано в прототип (__proto__) создаваемого объекта.
