const and let

First must undertand scoping in JS. First article and second one have good explanation about scope in JS. Historically it was defined as global and function scope. var made variable global if outside of function or global to function in inside of function. const and let are block scoped {}.

String templates

String templates are made with backticks and are one of basic types of javascript. In string templates string formatting is preserved. No need to type \n for new line. Here is good discussion on Stackoverflow about difference between single quoted and double quoted strings.
