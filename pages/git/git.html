Как добавить файлы в коммит? git add -A (--no-ignore-removal) добавит к комиту удалённые файлы.

Как добавить чанки?

Почему нельзя добавлять файлы по звезде или точке? Есть опастность добавит файлы которые были удалены из рабочего дерева.

Пуш. Запись в удаленный репозеторий.

Чем опасен пуш с форсом? Если кто-то уже сделал изменемия в бранч они будут потеряны.

Что такое фастфовард? По сути ничего не меняется просто HEAD будет указывать на коммит.

Как создать и удалить ветку, локально и в origin`е. Local: git branch [branch-name], git branch -d [branch-name]
Remote: git push origin [branch-name] or git push origin [branch-name]:[new branch-name] to delete git push origin :[branch-name] пробел обязателен!

Как привязать локальную ветку "a1" к удаленной ветке "b2", и пушить в нее? git branch -u a1 origin/b2, если не давать имя то дефолт на текущую.

Как получить список комитов из локальной ветки, которую вы только что удалили? git log --branch [name]

Сколько веток может указывать на один коммит? 'branches are free' не совсем но я думаю сколько угодно веток может указывать на какой-то коммит.

Что такое мердж? Как его откатить? git reset --merge

Какие бывают стратегия мерджа? resolve, recursive, octopus, ours, subtree

Что такое ребейз? Сдвиг истории всей ветки на другую ветку, история выглядит так, как будто изменения были в одной ветке. И позволяет переносить например готовые части в основную ветку. Do not rebase commits that exist outside your repository.
